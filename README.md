An AI-powered workplace productivity dashboard designed to help professionals automate everyday tasks using artificial intelligence. This application combines multiple AI tools into one modern and responsive platform to improve efficiency, communication, planning, and research in the workplace.
Features
✉️ Smart Email Generator
Generate professional emails in different tones such as formal, friendly, and persuasive.
📝 Meeting Notes Summarizer
Summarize meeting notes and extract key decisions, action items, and deadlines.
📅 AI Task Planner
Create structured daily or weekly schedules with prioritized tasks and productivity suggestions.
🔎 AI Research Assistant
Summarize topics, generate insights, and provide practical recommendations.
🤖 AI Chat Assistant
Interactive workplace AI chatbot for general productivity support and task assistance.
